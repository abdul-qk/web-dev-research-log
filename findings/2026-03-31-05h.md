# Web Dev Research – 5am Scan – 2026-03-31

## Major Announcements

- **Anthropic acquires Bun** (Dec 2025, ongoing impact): Anthropic acquired Bun — the all-in-one JS toolkit — as its first-ever acquisition. Bun remains MIT-licensed. Strategic rationale: Claude Code ships as a Bun executable; Bun is now core infrastructure for Claude Code, Claude Agent SDK, and future AI coding tools. Claude Code hit $1B ARR milestone around the same time.
- **Next.js 16.2 released** (March 2026): ~400% faster dev server startup vs 16.1. Key features: Server Fast Refresh (fine-grained server-side HMR via Turbopack), Hydration Diff Indicator in error overlay, stable Adapter API (cross-platform: Netlify, Cloudflare, AWS Amplify, OpenNext), AI-ready `AGENTS.md` default in `create-next-app`, Browser Log Forwarding to terminal, Experimental Agent DevTools for React DevTools access by AI agents. 200+ bug fixes.
- **Vite 8.0 released** (March 12, 2026): Most significant architectural change since Vite 2. Replaced dual-bundler system (esbuild for dev + Rollup for prod) with a single unified Rolldown (Rust-based) bundler. 10–30x faster production builds. Real-world: Linear's build time dropped from 46s to 6s. Also adds native TypeScript path aliases and unified dev/prod pipeline.

## Notable Releases

- **Flutter 3.41** (Feb 2026, HN trending today): 868 commits from 145 contributors. Highlights: experimental popup/tooltip window APIs, cross-platform dialog windows (Linux/macOS/Windows), merged threads on Linux by default, Swift Package Manager (SPM) support replacing CocoaPods for iOS plugins, modern iOS UIScene lifecycle by default, Material Design 3 integration, reduced shader compilation stutter.
- **Deno 2.6**: Introduced `dx` (like `npx` for npm/JSR binaries), `--unstable-tsgo` flag integrating TypeScript's native Go port for ~2x faster type checking, `deno audit`, native source maps, more granular permissions, source phase imports.
- **Prisma 7**: Complete query engine rewrite — ditched Rust engine for TypeScript/WebAssembly Query Compiler (~1.6MB / 600KB gzipped). Results: up to 3.4x faster queries on large result sets, up to 9x faster serverless cold starts. Also: multi-file schema support now stable (Prisma ≥6.6).
- **Remix 3**: No longer React-based — ground-up rethink of full-stack web dev, with AI playing a role in its design and implementation.
- **Tailwind Plus Elements**: New UI library from the Tailwind team built on native web platform features (`<dialog>`, popover API, custom elements) — drop-in script tag usage.

## Breaking Changes / Deprecations

- **Vite 8**: The dual-bundler architecture (esbuild + Rollup) is gone. Migration needed for any custom Rollup plugin configs not covered by Rolldown compatibility layer — though Vite team claims full plugin compatibility.
- **Flutter 3.41**: CocoaPods support being phased out in favor of Swift Package Manager for iOS plugins. Apps relying on CocoaPods-only plugins may need plugin updates.
- **Deno 2.6 tsgo**: `--unstable-tsgo` is experimental; some TypeScript edge cases may behave differently under the Go-based checker.
- **TypeScript 7 (preview)**: Native (Go port) previews are stable in editors. Close to 10x speedup over TS 6 without `--incremental`. Both TS 6.0 and 7.0 targeting early 2026 — watch for potential breaking type-system changes in 7.0.

## Emerging Trends

- **AI-first developer tooling**: Next.js 16.2's AGENTS.md, Browser Log Forwarding, Agent DevTools, and Anthropic's Bun acquisition all point to frameworks and runtimes being explicitly designed for AI coding agents as first-class users.
- **Rust-based build tooling dominance**: Rolldown (Vite 8), Turbopack (Next.js), Biome — the shift to Rust for JS tooling is now mainstream, not experimental. 10-30x build speed gains are becoming table stakes.
- **WebAssembly in production infrastructure**: Prisma 7's WASM query compiler, Blazor/Tailwind WASM integration — WASM is moving from browser novelty to server-side/tooling runtime (echoed in HN trending: "WebAssembly Beyond the Browser: The Universal Runtime Quietly Eating Software").
- **Platform-agnostic deployment**: Next.js 16.2's stable Adapter API (via OpenNext) signals growing pushback against Vercel lock-in; cross-platform portability becoming a framework-level concern.
- **TypeScript native speed race**: Go-port (tsgo via Deno/Microsoft) nearing 10x speedup; this will reshape editor latency and CI type-check times significantly when TS 7 ships.
